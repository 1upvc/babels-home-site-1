Google is adopting an autonomous defense model by embedding specialized AI agents across the entire software development lifecycle to counter machine-speed threats. The Mantis framework uses multi-agent orchestration and a hierarchical summary tree for context-aware code scanning, while self-healing fuzz testing automates harness creation and bug fixing. A unified pipeline then routes validated fixes from automated reproduction and regression tests directly to human reviewers.
